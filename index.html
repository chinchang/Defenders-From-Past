DEFENDERS FROM PAST

Game by Kushagra Gour (@chinchang457) for BYOG

Instructions:

Click on the green patches on the game
field to deploy Einstein to stop the PI
monsters from reaching the Pig king.

Make you placements wisely.

Play with the source at:

https://github.com/chinchang/Defenders-From-Past

- Platform
- Canvas
- engine
- very much similar to flash
- collision: rectangle based
- mouseevents
- Rendering
- images
- drawing api
- uses localstorage for the scores
- has debug mode
